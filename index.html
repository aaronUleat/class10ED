SASS

Es un preprocesador CSS que permite escribir codigo CSS aprovechando la funcionalidad de un lenguaje logico, con funciones, variables, scope, condicionales etc...

Nesting

Los selector pueden ser anidandos dentro de selectores, como decendientes

Partials and imports

Son los archivos que no deben compilarse por que se importarar dentro de otros, por convension se nombran con guion bajo _

@import "modules/main-menu"

Extend and placehoders

Extend es para heredar propiedades de otra regla.

	.sociales {
		display: inline-block
		line-height: 3rem;
		padding: 0 2em;
	}
	.sociales--footer {
		@extend .sociales;
		margin-top: 20px;
		font-family: Georgia;
	}

El placehoder es un pedazo de codigo reutilizable, a diferencia de un mixin un placeholder no recibi parametros

	%button { // Este es el placeholder
		display: inline-block;
		line-height: 2;
		padding: 0 1em;
	}
 
	button {
	  @extend %button;
	}
